Para criar um usuario:
    1-Digite um nome
    2-Digite uma senha
    3-Guarde o ID -------- o ID SEMPRE SERÁ O TIPO DE USUARIO("D" = GRADUANDO... "A" = PROFESSOR) + SUA POSICAO NA LISTA DE USUARIOS DA
                            BIBLIOTECA. EX: "D0","C1","B2", ETC...

Para criar um material:
    1- Digite as informações
    2- Guarde o ID do Material(Ou apenas veja o id dele na lista de materiais da biblioteca apertando "7")

Para realizar um emprestimo
    1- Digite o ID
    2- Digite a senha
    3- Digite o ID do material

Para realizar uma devolução
    1- Digite o ID
    2- Digite a senha
    3- Digite o ID do material
